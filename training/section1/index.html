Four JavaScript Facts

JavaScript Fact #1

JavaScript is a programming language for your browser. It is interpreted instead of compiled.

JavaScript Fact #2

JavaScript Fact #3

Each browser and device has its own implementation of JavaScript, but the ECMAScript specification is what guides JavaScript enhancements in each browser.

JavaScript Fact #4

JavaScript is always updating.

Script Tags

You can mix JavaScript and HTML. The script tag tells your browser the stuff inside is code, not content.

   // CODE GOES HERE
</script>

It's time to talk about the DOM.

The Document Object Model (DOM)

The DOM is a programming interface for HTML documents.

The Document Object Model (DOM)

It represents the page so that programs can change the document structure, style, and content.

The Document Object Model (DOM)

The DOM represents the document as nodes and objects.

The DOM: Sample Code


<!DOCTYPE html>
<html>
  <head>
    <title>Test Page</title>
    <style>
      h1 {
        color: red;
      }
    </style>
  </head>
  <body>
    <h1>My Page</h1>
    <p>Hello World!</p>
    <img src="img.png" />
  </body>
</html>

What does it mean that there's a programming interface?

There is a list of commands that allow for access, creation, and modification.

Modified via the DOM

The key to the DOM is targeting the right element.

tag, id and class


   <h1 id="headline">Breaking News!</h1>
   <p>
     Traffic is <span class="warning">backed</span> up in the
     Fort Pitt Tunnel!
   </p>

What is the difference between id and class?

id

Use only once on a page.

Why only once?

It's a specific target.

Also used for anchor links.

https://developer.mozilla.org/en-US/docs/Web/JavaScript#Tutorials

class

Use as many times as you would like.

DOM Access: By Tag Name


  // document.getElementsByTagName(tagName) - get list;
  document.getElementsByTagName("a");

  // document.getElementsByTagName(tagName)[number] - get 1;
  document.getElementsByTagName("a")[0];

Get List of tags

document.getElementsByTagName("img");

DOM Access: By Class Name


	// document.getElementsByClassName(class) - get list;
	document.getElementsByClassName("warning");

	// document.getElementsByClassName(tagName)[number] - get 1;
	document.getElementsByClassName("warning")[0];

Get List of tags

document.getElementsByClassName("rISBZc");

DOM Access: By ID


	// document.getElementById(id) - get 1;
	document.getElementById("headline");

Get a single element

document.getElementById("dimg_6");

Watch out!

document.getElementById vs document.getElementsByClassName

The "s" is very important

Now what?

If you can target an element.

You can change an element.


  document.getElementsByTagName("a")[0].innerHTML = "Nailed it!";

Change the style

document.getElementsByClassName("e2BEnf")[0].innerHTML="<h3>COFFEE!</h3>";
