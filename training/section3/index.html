Arrays

Arrays are lists that contain values.

You can put different types of data into an array.

              
    let rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
    'Blue', 'Indigo', 'Violet'];
    let lotteryNumbers = [33, 72, 64, 18, 17, 85];
    let myFavoriteThings = ['family', 3, 'guitars', 'woodworking'];

What are arrays used for?

Arrays are lists of information.

How do they work?

						
  // DECLARATION
  let rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
  'Blue', 'Indigo', 'Violet'];

  // GET VALUE
  rainbowColors[3]; // Green

Array.length

How many items are in the list?

.length

							
  // DECLARATION
  let rainbowColors = ['Red', 'Orange', 'Yellow', 'Green',
  'Blue', 'Indigo', 'Violet'];

  // GET VALUE
  rainbowColors.length; // 7

For loops repeat the same code over and over until a condition is met.

  console.log(i);
}
